# Excel 工具更新说明

## 版本 1.0.0 (2024-03-02)

### 新增功能

#### 1. Excel 读取器 (ExcelReader)
- 支持读取 .xlsx 格式的 Excel 文件
- 支持 4 行格式：注释行、字段名行、类型定义行、数据行
- 支持多种数据类型解析：
  - 基础类型：int, long, float, double, string, bool
  - 数组类型：int[], float[], string[]
  - 枚举类型
  - 自定义类型（通过 CustomTypeParser 或 Parse 方法）
- 自动跳过空行和空字段

#### 2. 自定义类型解析系统 (ICustomTypeParser)
- 三种解析优先级：
  1. CustomTypeParser 特性
  2. 静态 Parse(string) 方法
  3. JSON 反序列化
- 内置常用类型解析器：
  - Vector2Parser, Vector3Parser
  - ColorParser
  - KeyValuePairParser
  - IntRangeParser
  - ItemRewardParser
  - RewardListParser
- 支持自定义解析器扩展

#### 3. 数据校验器 (DataValidator)
- 主键重复检查
- 类型检查
- 外键检查（ForeignKey 特性）
- 范围校验（Range 特性）
- 详细的错误和警告信息

#### 4. 代码生成器 (CodeGenerator)
- 根据 Excel 表结构生成 C# 配置类
- 生成配置表加载类
- 支持 SQLite 特性（Table, PrimaryKey, Column）
- 支持 Serializable 特性
- 自动生成注释

#### 5. 配置表编辑器 (ConfigTableEditorWindow)
- 两种工作模式：
  - LoadAndGenerate：从现有 Excel 加载并生成代码
  - CreateTemplate：定义字段并导出模板说明
- 可视化字段编辑：
  - 主键标记
  - 字段名、类型、注释
  - 字段顺序调整（上移/下移）
  - 字段删除
- 导出模板说明文件（.txt）

#### 6. Excel 导出器 (ExcelExporter)
- 单个 Excel 文件导出到 SQLite
- 批量 Excel 文件导出
- 导出进度显示
- 导出错误日志
- 支持覆盖/不覆盖已存在的表
- 详细日志输出（可选）
- 基本数据校验（主键重复检查）

### 技术特性

#### 分隔符使用规范
- 逗号 (,): 同级元素分隔（坐标、颜色、数组）
- 冒号 (:): 键值对
- 分号 (;): 记录/组分隔
- 竖线 (|): 复杂结构字段
- 星号 (*): 权重
- 减号/波浪 (-/~): 范围

#### Excel 格式规范
```
行0: 注释行（字段说明）
行1: 字段名行
行2: 类型定义行
行3+: 数据行
```

### 使用方式

#### 方式一：使用可视化工具

1. **配置表编辑器**
   - 菜单：`Tools > Excel > 配置表编辑器`
   - 用于创建新配置表或修改现有配置表

2. **Excel 导出器**
   - 菜单：`Tools > Excel > Excel 导出器`
   - 用于将 Excel 数据导出到 SQLite 数据库

#### 方式二：使用代码

```csharp
// 1. 读取 Excel
var reader = new ExcelReader();
var sheets = reader.ReadExcel("path/to/config.xlsx");

// 2. 生成代码
var generator = new CodeGenerator();
var code = generator.GenerateConfigClass(sheets[0], "ItemConfig");

// 3. 导出到 SQLite
var exporter = new ExcelExporter(new ExcelExporter.ExportConfig
{
    OutputDbPath = "Assets/StreamingAssets/RefData/config.db",
    OverwriteExistingTables = true,
    EnableValidation = true
});
var result = exporter.ExportExcel("path/to/config.xlsx");
```

### 完整工作流程

```
1. 使用配置表编辑器定义配置表结构
   ↓
2. 导出模板说明文件
   ↓
3. 使用 Excel 工具按照模板创建 Excel 文件
   ↓
4. 使用配置表编辑器加载 Excel 并生成 C# 代码
   ↓
5. 在 Excel 中填充数据
   ↓
6. 使用 Excel 导出器将数据导出到 SQLite
   ↓
7. 在游戏中加载配置表
```

### 已知限制

1. **不支持直接创建/修改 Excel 文件**
   - 需要使用外部工具（Microsoft Excel、WPS 等）

2. **数据校验功能有限**
   - 导出器目前只支持主键重复检查
   - 完整的数据校验需要使用 DataValidator（需要类型信息）

3. **SQLite 库**
   - 使用 sqlite-net-pcl 而不是 System.Data.SQLite
   - 部分高级功能可能不支持
